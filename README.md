# create-password-function

## Usage

```javascript
const CreatePasswordFunction = require("./create-password-function.js")
const password = client.CreatePasswordFunction({
  lenght: 12,
  number: true,
  symbol: true,
  lowercaseLetter: true,
  uppercaseLetter: true
})

console.log(password)
```
